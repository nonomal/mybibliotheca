Data Export Options

Export your library data in various formats for backup, analysis, or migration to other systems.

Export your complete library as a CSV file. Compatible with Excel, Google Sheets, and most other applications.

JSON Export

  • Complete data structure preservation
  • Includes all relationships and metadata

  • Ideal for data migration

Export Information
CSV Format

The CSV export includes these columns:

  • Title
  • Author(s)
  • ISBN13 & ISBN10
  • Published Year
  • Reading Status
  • Start & Finish Dates
  • Personal Rating
  • Personal Notes
  • Publisher
  • Page Count
  • Categories
JSON Format

The JSON export preserves the complete data structure including nested relationships and metadata.

Privacy Note: Exported data includes all your personal information such as reading dates, ratings, and notes. Keep exported files secure.
Usage Tips
For Migration

Use JSON export when moving to another system that can import structured data.

For Analysis

Use CSV export to analyze your reading habits in Excel, Google Sheets, or data analysis tools.

For Sharing

CSV format is more universally compatible for sharing reading lists.

For Backup

Use the full backup system for complete data protection and restoration capabilities.
